# 阶段三：全链路小型 Capstone — 模板套件

## B. 技术方案（简版）

```markdown
- 模块：是否动 `wmt-module-*` / 框架 starter
- 接口：新增/变更的 REST 路径（含 `/admin-api` 前缀）
- 数据：表变更、迁移脚本编号

## 关键设计
- 并发与事务：
- 缓存与一致性：
- 安全与权限：

## 测试策略
- 单元 / WebMvcTest / 集成 / 手工

## 上线与回滚
- 开关（feature flag）：
- 回滚步骤：
```

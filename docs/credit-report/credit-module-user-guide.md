## 快速开始

### 系统角色说明

征信模块包含两种主要角色：

| 角色 | 权限说明 | 主要功能 |
|------|---------|---------|
| **管理员** | 系统配置权限 | 配置字段、配置计算规则、生成汇总报表、查看所有数据 |
| **表单专员** | 部门数据权限 | 填写本部门表单、提交表单、查看本部门历史记录 |

### 首次使用流程

```
管理员首次使用：
1. 登录系统 → 2. 配置字段 → 3. 配置计算规则 → 4. 等待表单专员填写 → 5. 生成汇总报表

表单专员首次使用：
1. 登录系统 → 2. 选择报送周期 → 3. 填写表单 → 4. 保存草稿 → 5. 提交表单
```

## 管理员操作指南

### 一、登录系统

1. 打开系统登录页面
2. 输入管理员账号和密码
3. 点击"登录"按钮
4. 登录成功后，进入系统主界面

**注意事项：**
- 管理员账号需要具有 `credit:field-config:*`、`credit:calculation-rule:*`、`credit:summary-report:*` 等权限
- 如果看不到征信模块菜单，请联系系统管理员分配权限

### 二、配置字段（首次使用必做）

字段配置是动态表单的基础，管理员需要先配置好各部门需要填写的字段。

#### 2.1 进入字段配置页面

1. 在主菜单中找到 **"征信管理"** → **"字段配置"**
2. 点击进入字段配置列表页面

#### 2.2 创建通用字段（所有部门共享）

**场景说明：** 通用字段是所有部门都需要填写的字段，例如"资产总额"、"负债总额"等。

**操作步骤：**

1. 点击页面右上角的 **"新增"** 按钮
2. 填写字段信息：
   - **部门编号**：选择 `0`（0表示通用字段）
   - **字段编码**：输入唯一标识，例如 `assets`（建议使用英文，不能有空格）
   - **字段名称**：输入显示名称，例如 `资产总额`
   - **字段类型**：选择字段类型
     - `NUMBER`：整数（如：员工数量）
     - `DECIMAL`：小数（如：金额，需要保留2位小数）
     - `TEXT`：文本（如：备注说明）
     - `DATE`：日期（如：成立日期）
     - `DATETIME`：日期时间（如：报送时间）
     - `BOOLEAN`：布尔值（如：是否启用）
   - **是否必填**：勾选表示该字段必须填写
   - **默认值**：输入默认值（可选）
   - **校验规则**：点击"编辑"按钮，配置校验规则
     - 数值类型：可设置最小值、最大值、小数位数
     - 文本类型：可设置最小长度、最大长度、正则表达式
     - 日期类型：可设置最小日期、最大日期
   - **显示顺序**：输入数字，数字越小越靠前显示
   - **状态**：选择"启用"
3. 点击 **"确定"** 保存

**示例：创建"资产总额"字段**

```
部门编号：0
字段编码：total_assets
字段名称：资产总额
字段类型：DECIMAL
是否必填：是
默认值：0
校验规则：
  - 最小值：0
  - 最大值：999999999.99
  - 小数位数：2
显示顺序：1
状态：启用
```

#### 2.3 创建部门特有字段

**场景说明：** 某些部门有特殊字段，只有该部门需要填写。

**操作步骤：**

1. 点击页面右上角的 **"新增"** 按钮
2. 填写字段信息：
   - **部门编号**：选择具体的部门（不是0）
   - 其他字段配置同通用字段
3. 点击 **"确定"** 保存

**示例：创建"财务部"特有字段"预算执行率"**

```
部门编号：财务部（ID=2）
字段编码：budget_execution_rate
字段名称：预算执行率
字段类型：DECIMAL
是否必填：是
校验规则：
  - 最小值：0
  - 最大值：100
  - 小数位数：2
显示顺序：10
状态：启用
```

#### 2.4 编辑字段配置

1. 在字段列表中，找到需要修改的字段
2. 点击该行的 **"编辑"** 按钮
3. 修改字段信息
4. 点击 **"确定"** 保存

**注意事项：**
- 修改字段配置后，已填写的表单数据不会自动更新
- 如果字段编码被修改，可能导致已填写的表单数据无法正确显示

#### 2.5 禁用/启用字段

1. 在字段列表中，找到需要操作的字段
2. 点击 **"编辑"** 按钮
3. 修改 **"状态"** 为"禁用"或"启用"
4. 点击 **"确定"** 保存

**注意事项：**
- 禁用的字段不会在表单中显示
- 已填写的表单数据中的禁用字段值仍然保留

#### 2.6 删除字段

1. 在字段列表中，找到需要删除的字段
2. 点击该行的 **"删除"** 按钮
3. 确认删除操作

**注意事项：**
- 删除字段前，请确认没有表单数据使用该字段
- 删除后，已填写的表单数据中的该字段值将无法显示

### 三、配置计算规则（首次使用必做）

计算规则用于自动计算汇总报表中的数据。

#### 3.1 进入计算规则配置页面

1. 在主菜单中找到 **"征信管理"** → **"计算规则配置"**
2. 点击进入计算规则列表页面

#### 3.2 创建计算规则

**场景说明：** 月报中"总资产" = 各部门"资产总额"之和

**操作步骤：**

1. 点击页面右上角的 **"新增"** 按钮
2. 填写计算规则信息：
   - **目标字段编码**：输入计算结果字段的编码，例如 `total_assets_sum`
   - **报表类型**：选择 `MONTHLY`（月报）或 `QUARTERLY`（季报）
   - **规则类型**：选择规则类型
     - `SUM`：求和（各部门字段值相加）
     - `AVG`：平均值
     - `FORMULA`：公式计算（使用SpEL表达式）
     - `AGGREGATE`：聚合（从历史报表获取数据）
   - **计算表达式**：输入SpEL表达式
     - SUM类型示例：`#deptFormData.values().![get('total_assets')].sum()`
     - FORMULA类型示例：`#deptFormData['1'].get('assets') + #deptFormData['2'].get('assets')`
   - **数据来源**：输入JSON格式的数据来源说明（可选）
   - **计算顺序**：输入数字，数字越小越先计算
   - **规则描述**：输入规则说明，例如"计算各部门资产总额之和"
   - **状态**：选择"启用"
3. 点击 **"确定"** 保存

**示例：创建月报"总资产"计算规则**

```
目标字段编码：total_assets_sum
报表类型：MONTHLY
规则类型：SUM
计算表达式：#deptFormData.values().![get('total_assets')].sum()
数据来源：{"type": "FORM_DATA", "fields": ["total_assets"]}
计算顺序：1
规则描述：计算各部门资产总额之和
状态：启用
```

**示例：创建季报"当期发生额"计算规则**

```
目标字段编码：current_period_occurrence
报表类型：QUARTERLY
规则类型：FORMULA
计算表达式：#latestMonthlyReport.get('field3') - #previousQuarter.get('cumulativeOccurrence')
数据来源：{
  "latestMonthlyReport": {"type": "MONTHLY_REPORT", "reportPeriod": "${quarterLatestMonth}"},
  "previousQuarter": {"type": "QUARTERLY_REPORT", "reportPeriod": "${previousQuarterPeriod}"}
}
计算顺序：2
规则描述：当期发生额 = 季度最后一个月报的field3 - 上季度累计发生额
状态：启用
```

#### 3.3 编辑计算规则

1. 在计算规则列表中，找到需要修改的规则
2. 点击该行的 **"编辑"** 按钮
3. 修改规则信息
4. 点击 **"确定"** 保存

#### 3.4 测试计算规则

**操作步骤：**

1. 确保已有表单数据提交
2. 进入汇总报表页面，点击 **"生成报表"**
3. 查看生成的报表数据，验证计算结果是否正确
4. 如果计算结果不正确，检查计算规则表达式是否正确

### 四、配置校验规则（可选）

校验规则用于在表单提交时进行数据校验。

#### 4.1 进入校验规则配置页面

1. 在主菜单中找到 **"征信管理"** → **"校验规则配置"**
2. 点击进入校验规则列表页面

#### 4.2 创建校验规则

**操作步骤：**

1. 点击页面右上角的 **"新增"** 按钮
2. 填写校验规则信息：
   - **字段编码**：选择需要校验的字段
   - **规则类型**：选择规则类型
     - `REQUIRED`：必填校验
     - `RANGE`：范围校验（最小值、最大值）
     - `REGEX`：正则表达式校验
     - `CUSTOM`：自定义校验
   - **校验表达式**：输入校验表达式
     - RANGE示例：`min:0,max:1000000`
     - REGEX示例：`^[0-9]+$`（只允许数字）
   - **错误提示信息**：输入校验失败时的提示信息
   - **优先级**：输入数字，数字越大优先级越高
   - **状态**：选择"启用"
3. 点击 **"确定"** 保存

**注意事项：**
- 字段配置中的"是否必填"和"校验规则"已经包含了基本的校验功能
- 校验规则配置主要用于更复杂的校验场景

---

### 五、生成汇总报表

#### 5.1 进入汇总报表页面

1. 在主菜单中找到 **"征信管理"** → **"汇总报表"**
2. 点击进入汇总报表列表页面

#### 5.2 生成月报

**前提条件：**
- 所有部门的月报表单都已提交（状态为"已提交"）

**操作步骤：**

1. 点击页面右上角的 **"生成报表"** 按钮
2. 填写生成参数：
   - **报送周期**：选择月份，格式为 `YYYY-MM`，例如 `2024-01`
   - **报表类型**：选择 `MONTHLY`（月报）
3. 点击 **"确定"** 开始生成
4. 系统开始计算，页面显示"计算中"状态
5. 等待计算完成（通常需要几秒到几分钟，取决于数据量和计算复杂度）
6. 计算完成后，状态变为"已完成"
7. 点击报表行的 **"查看"** 按钮，查看汇总数据

**注意事项：**
- 如果某个部门的表单未提交，生成报表会失败，提示"存在未提交的表单"
- 生成报表是异步操作，可以关闭页面，系统会在后台继续计算
- 如果计算失败，可以查看错误日志，修复问题后重新生成

#### 5.3 生成季报

**前提条件：**
- 季度内所有月份（3个月）的月报都已生成
- 如果需要计算累计数据，上季度的季报也需要已生成

**操作步骤：**

1. 点击页面右上角的 **"生成报表"** 按钮
2. 填写生成参数：
   - **报送周期**：选择季度，格式为 `YYYY-Q1/Q2/Q3/Q4`，例如 `2024-Q1`
   - **报表类型**：选择 `QUARTERLY`（季报）
3. 点击 **"确定"** 开始生成
4. 等待计算完成
5. 查看汇总数据

**季度说明：**
- Q1：1-3月
- Q2：4-6月
- Q3：7-9月
- Q4：10-12月

#### 5.4 查看汇总报表详情

1. 在汇总报表列表中，找到需要查看的报表
2. 点击该行的 **"查看"** 按钮
3. 查看报表数据：
   - 报表基本信息：报送周期、报表类型、生成时间、状态
   - 报表数据：以表格形式展示所有汇总字段的值
   - 计算日志：显示计算过程（如果有）

#### 5.5 导出Excel

1. 在汇总报表列表中，找到需要导出的报表
2. 点击该行的 **"导出Excel"** 按钮
3. 系统自动下载Excel文件
4. 打开Excel文件查看数据

**注意事项：**
- Excel文件包含报表的所有数据
- 表头使用字段名称（不是字段编码）
- 如果报表数据较多，导出可能需要几秒钟

### 六、查看表单数据（管理员）

管理员可以查看所有部门的表单数据。

#### 6.1 查看表单列表

1. 在主菜单中找到 **"征信管理"** → **"表单数据"**
2. 进入表单数据列表页面
3. 可以按以下条件筛选：
   - **部门**：选择部门
   - **报送周期**：选择月份或季度
   - **报表类型**：选择月报或季报
   - **状态**：选择状态（草稿、已提交、已报送）
4. 点击 **"查询"** 按钮筛选数据

#### 6.2 查看表单详情

1. 在表单列表中，找到需要查看的表单
2. 点击该行的 **"查看"** 按钮
3. 查看表单数据：
   - 基本信息：部门、报送周期、报表类型、状态、提交人、提交时间
   - 表单数据：以表格形式展示所有字段的值

#### 6.3 查看审计日志

1. 在主菜单中找到 **"征信管理"** → **"审计日志"**
2. 进入审计日志列表页面
3. 可以按以下条件筛选：
   - **业务类型**：选择业务类型（表单数据、汇总报表）
   - **业务编号**：输入业务编号
   - **操作类型**：选择操作类型（创建、更新、提交、计算）
   - **操作人**：选择操作人
   - **操作时间**：选择时间范围
4. 点击 **"查询"** 按钮筛选日志
5. 点击日志行的 **"查看"** 按钮，查看详细信息：
   - 操作信息：操作类型、操作人、操作时间、IP地址
   - 变更内容：变更前数据、变更后数据、变更字段列表

## 表单专员操作指南

### 一、登录系统

1. 打开系统登录页面
2. 输入账号和密码
3. 点击"登录"按钮
4. 登录成功后，进入系统主界面

**注意事项：**
- 表单专员账号需要具有 `credit:form-data:*` 权限
- 表单专员只能查看和操作本部门的表单数据

---

### 二、填写表单

#### 2.1 进入表单填报页面

1. 在主菜单中找到 **"征信管理"** → **"表单填报"**
2. 点击进入表单填报页面

#### 2.2 选择报送周期

1. 在页面顶部，选择 **"报送周期"**
   - 月报：选择月份，格式为 `YYYY-MM`，例如 `2024-01`
   - 季报：选择季度，格式为 `YYYY-Q1/Q2/Q3/Q4`，例如 `2024-Q1`
2. 选择 **"报表类型"**：`MONTHLY`（月报）或 `QUARTERLY`（季报）
3. 点击 **"开始填报"** 按钮

**注意事项：**
- 如果该周期已有表单数据，系统会自动加载已保存的数据
- 如果该周期已有表单已提交，表单将变为只读模式，无法修改

#### 2.3 填写表单字段

表单会根据字段配置自动生成，包含以下类型的字段：

**数值字段（NUMBER/DECIMAL）：**
- 直接输入数字
- 小数类型会自动限制小数位数
- 必填字段会有红色星号（*）标记

**文本字段（TEXT）：**
- 直接输入文本
- 可以输入多行文本（如果是多行文本框）

**日期字段（DATE）：**
- 点击日期选择器，选择日期
- 或直接输入日期，格式为 `YYYY-MM-DD`

**布尔字段（BOOLEAN）：**
- 勾选复选框表示"是"
- 不勾选表示"否"

**填写技巧：**
- 必填字段必须填写，否则无法保存
- 输入时会有实时校验提示
- 鼠标移出输入框时，会进行字段校验
- 如果字段值不符合校验规则，会显示红色错误提示

**示例：填写"资产总额"字段**

```
1. 找到"资产总额"字段（显示顺序为1，通常在最上方）
2. 在输入框中输入数值，例如：1000000.00
3. 如果输入错误（如负数或超过最大值），会显示错误提示
4. 修正错误后，错误提示消失
```

#### 2.4 保存草稿

**操作步骤：**

1. 填写完部分字段后，点击页面底部的 **"保存草稿"** 按钮
2. 系统提示"保存成功"
3. 表单数据已保存，可以随时继续填写

**注意事项：**
- 草稿状态下的表单可以随时修改
- 草稿不会参与汇总计算
- 建议定期保存草稿，避免数据丢失

#### 2.5 提交表单

**前提条件：**
- 所有必填字段都已填写
- 所有字段值都符合校验规则

**操作步骤：**

1. 检查表单数据是否完整和正确
2. 点击页面底部的 **"提交"** 按钮
3. 系统会进行最终校验
4. 如果校验通过，系统提示"提交成功"
5. 表单状态变为"已提交"，无法再修改

**注意事项：**
- 提交后的表单无法修改，如需修改请联系管理员
- 提交后的表单会参与汇总计算
- 提交前请仔细核对数据，确保准确无误

**提交失败处理：**

如果提交失败，系统会显示错误信息：
- **"存在必填字段未填写"**：检查是否有必填字段为空
- **"字段值不符合校验规则"**：检查字段值是否在允许范围内
- **"该周期表单已提交"**：该周期已有表单已提交，无法重复提交

### 三、查看历史记录

#### 3.1 查看表单列表

1. 在主菜单中找到 **"征信管理"** → **"我的表单"**
2. 进入表单列表页面
3. 可以按以下条件筛选：
   - **报送周期**：选择月份或季度
   - **报表类型**：选择月报或季报
   - **状态**：选择状态（草稿、已提交、已报送）
4. 点击 **"查询"** 按钮筛选数据

**注意事项：**
- 表单专员只能查看本部门的表单数据
- 其他部门的表单数据不会显示

#### 3.2 查看表单详情

1. 在表单列表中，找到需要查看的表单
2. 点击该行的 **"查看"** 按钮
3. 查看表单数据：
   - 基本信息：报送周期、报表类型、状态、提交时间
   - 表单数据：以表格形式展示所有字段的值

**注意事项：**
- 已提交的表单为只读模式，无法修改
- 草稿状态的表单可以点击"编辑"按钮继续填写

#### 3.3 编辑草稿表单

1. 在表单列表中，找到状态为"草稿"的表单
2. 点击该行的 **"编辑"** 按钮
3. 修改表单数据
4. 点击 **"保存草稿"** 或 **"提交"** 按钮

## 常见问题解答

### 管理员常见问题

#### Q1: 如何知道哪些字段是通用字段，哪些是部门字段？

**A:** 在字段配置列表中，查看"部门编号"列：
- 部门编号为 `0` 的是通用字段（所有部门都显示）
- 部门编号为具体数字的是部门字段（只有该部门显示）

#### Q2: 修改字段配置后，已填写的表单数据会受影响吗？

**A:** 
- 如果只是修改字段名称、显示顺序等显示相关的内容，不会影响已填写的表单数据
- 如果修改字段编码，已填写的表单数据中的该字段值将无法正确显示
- 如果禁用字段，已填写的表单数据中的该字段值仍然保留，但不会在表单中显示

#### Q3: 生成汇总报表失败，提示"存在未提交的表单"，怎么办？

**A:** 
1. 进入"表单数据"页面
2. 筛选该报送周期的表单数据
3. 查看哪些部门的表单状态为"草稿"
4. 联系对应的表单专员，提醒他们提交表单
5. 所有表单都提交后，重新生成报表

#### Q4: 计算规则表达式怎么写？

**A:** 
- 计算规则使用SpEL（Spring Expression Language）表达式
- SUM类型示例：`#deptFormData.values().![get('field_code')].sum()`
- FORMULA类型示例：`#deptFormData['1'].get('assets') + #deptFormData['2'].get('assets')`
- 详细语法请参考SpEL文档或联系技术支持

#### Q5: 如何查看计算日志？

**A:** 
1. 进入汇总报表详情页面
2. 查看"计算日志"部分
3. 计算日志会显示每个字段的计算过程和结果

### 表单专员常见问题

#### Q1: 为什么我看不到某些字段？

**A:** 
可能的原因：
1. 该字段是其他部门的特有字段，您看不到是正常的
2. 该字段已被管理员禁用
3. 该字段的报表类型与当前选择的报表类型不匹配

#### Q2: 保存草稿后，数据会丢失吗？

**A:** 
不会。保存草稿后，数据会永久保存在系统中，可以随时继续填写。

#### Q3: 提交表单后，还能修改吗？

**A:** 
不能。提交后的表单无法修改。如果需要修改，请联系管理员。

#### Q4: 如何知道我的表单是否已提交？

**A:** 
1. 在"我的表单"列表中，查看"状态"列
2. 状态为"草稿"表示未提交
3. 状态为"已提交"表示已提交
4. 状态为"已报送"表示已参与汇总计算

#### Q5: 为什么我无法提交表单？

**A:** 
可能的原因：
1. 存在必填字段未填写（检查是否有红色星号标记的字段）
2. 字段值不符合校验规则（检查是否有红色错误提示）
3. 该周期表单已提交（检查是否已有该周期的表单）

#### Q6: 如何查看汇总报表？

**A:** 
1. 进入"汇总报表"页面
2. 筛选需要查看的报送周期和报表类型
3. 点击"查看"按钮查看报表详情
4. 或点击"导出Excel"按钮导出报表

## 操作流程图

### 管理员完整流程

```
开始
  ↓
登录系统
  ↓
配置字段（首次使用）
  ├─→ 创建通用字段
  └─→ 创建部门字段
  ↓
配置计算规则（首次使用）
  ├─→ 创建月报计算规则
  └─→ 创建季报计算规则
  ↓
等待表单专员提交表单
  ↓
生成汇总报表
  ├─→ 生成月报
  └─→ 生成季报
  ↓
查看汇总报表
  ├─→ 查看报表详情
  └─→ 导出Excel
  ↓
结束
```

### 表单专员完整流程

```
开始
  ↓
登录系统
  ↓
选择报送周期
  ├─→ 选择月份（月报）
  └─→ 选择季度（季报）
  ↓
填写表单字段
  ├─→ 填写必填字段
  └─→ 填写可选字段
  ↓
保存草稿（可选）
  ↓
提交表单
  ├─→ 校验通过 → 提交成功
  └─→ 校验失败 → 修正错误 → 重新提交
  ↓
查看汇总报表（可选）
  ↓
结束
```

### 月报生成流程

```
管理员点击"生成月报"
  ↓
检查所有部门表单是否已提交
  ├─→ 否 → 提示"存在未提交的表单" → 结束
  └─→ 是 → 继续
  ↓
加载月报计算规则
  ↓
从各部门表单获取数据
  ↓
执行计算规则
  ├─→ SUM类型：求和
  ├─→ AVG类型：平均值
  ├─→ FORMULA类型：公式计算
  └─→ AGGREGATE类型：聚合计算
  ↓
保存汇总报表数据
  ↓
状态更新为"已完成"
  ↓
结束
```

### 季报生成流程

```
管理员点击"生成季报"
  ↓
检查季度内所有月报是否已生成
  ├─→ 否 → 提示"月报尚未生成" → 结束
  └─→ 是 → 继续
  ↓
检查上季度季报是否已生成（如果需要）
  ├─→ 否 → 提示"上季度季报尚未生成" → 结束
  └─→ 是 → 继续
  ↓
加载季报计算规则
  ↓
从月报和上季度季报获取数据
  ↓
执行计算规则
  ↓
保存汇总报表数据
  ↓
状态更新为"已完成"
  ↓
结束
```
